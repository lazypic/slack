# Slack
Lazypic은 채팅 솔루션으로 Slack을 사용합니다.
Slack자체를 그대로 사용가능하지만 조금더 커스터마이징 하기위해서
셋팅하는 방법을 다루는 리포지터를 생성하였습니다.

설정페이지 : https://lazypic.slack.com/customize
